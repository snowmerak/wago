# wago

`wago` is a developer toolchain helper for Go WebAssembly (WASM). It automates the generation of type-safe conversion bindings between Go structs and JavaScript classes, and provides a unified build pipeline.

## Features

- **Automated Mapping**: Generates `ToJSValue` and `FromJSValue` conversion methods for Go structs, alongside equivalent ES6 JavaScript classes with JSDoc type annotations.
- **Subpackage Layout Preservation**: Automatically stages generated JS assets inside the output directory (e.g. `dist/`), mirroring the original Go package directory structure.
- **WASM Build Integration**: `wago build` automatically runs generators, compiles the binary targeting WebAssembly (`GOOS=js GOARCH=wasm`), and extracts the corresponding `wasm_exec.js` from the local Go SDK.

## Installation

Compile and install the CLI tool:

```bash
go install wago@latest
```

Or build from source:

```bash
go build -o wago main.go
```

## Usage

### 1. Code Generation

Add the `//go:generate` directive above Go struct definitions:

```go
package main

//go:generate wago -type=User,Profile

type User struct {
	Name    string
	Age     int
	Profile Profile
}

type Profile struct {
	Bio string
}
```

Running `go generate` directly or via the build command will output:
1. **Go Mapper (`[file]_wago.go`)**:
   - `ToJSValue() js.Value` method on target structs.
   - `[TypeName]FromJSValue(js.Value) [TypeName]` helper function.
2. **JS Mapper (`[file].js`)**:
   - ES6 classes matching the structures with `constructor`, static `fromJS(obj)`, and `toJS()` methods.

### 2. Unified Build

To run generation, build the WebAssembly binary, and stage frontend assets in one command:

```bash
wago build [-o dist/main.wasm] [extra go build args...]
```

#### Output Structure

By default, files are generated and staged under the `dist/` directory:

```
dist/
├── main.wasm
├── wasm_exec.js
└── test/
    └── test_struct.js   (Staged with package layout preserved)
```

## Generated Mappings Example

### Go Source

```go
type User struct {
	Name string
	Age  int
}
```

### Generated Go Mapper (`user_wago.go`)

```go
//go:build js && wasm

// Code generated by wago; DO NOT EDIT.

package main

import "syscall/js"

func (u User) ToJSValue() js.Value {
	obj := js.Global().Get("Object").New()
	obj.Set("name", js.ValueOf(u.Name))
	obj.Set("age", js.ValueOf(u.Age))
	return obj
}

func UserFromJSValue(val js.Value) User {
	var u User
	if val.IsUndefined() || val.IsNull() {
		return u
	}
	if temp := val.Get("name"); !temp.IsUndefined() && !temp.IsNull() {
		u.Name = temp.String()
	}
	if temp := val.Get("age"); !temp.IsUndefined() && !temp.IsNull() {
		u.Age = int(temp.Int())
	}
	return u
}
```

### Generated JavaScript Class (`user.js`)

```javascript
// Code generated by wago; DO NOT EDIT.

/**
 * @property {string} name
 * @property {number} age
 */
export class User {
	/**
	 * @param {string} name
	 * @param {number} age
	 */
	constructor(name, age) {
		this.name = name;
		this.age = age;
	}

	/**
	 * @param {object} obj
	 * @returns {User|null}
	 */
	static fromJS(obj) {
		if (!obj) return null;
		return new User(
			obj.name,
			obj.age
		);
	}

	/**
	 * @returns {object}
	 */
	toJS() {
		return {
			name: this.name,
			age: this.age
		};
	}
}
```
